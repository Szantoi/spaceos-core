# JOINERY-PLANNING-V2 — Phase 1: Gyártásilap PDF

## Overview

**Scope:** PDF generation for manufacturing specification sheet (gyártásilap) — 4 label variants + supplier order list

**Why:** Doorstar production flow — operators print gyártásilap + labels at cutting workstation, assembly instructions + material list

**Phase 1 Timeline:** 5-7 days (1 FTE)

### QuestPDF Integration

- **NuGet:** QuestPDF 2024.10 + Community License

- **Application:** `Joinery.Application.Queries.GenerateGyartasila/` + `Joinery.Application.Commands.StoreGyartasila/`

    // - Archive() → Status = Archived

public enum GyartasilapStatus { Draft, Finalized, Archived }

public record LabelVariant(
    string Code,           // "L1", "L2", "L3", "L4"
    string Description,    // "Basic", "Premium", "Barcode", "QR"
    bool IncludeBarcode,
    bool IncludeQrCode,
    int FontSizePt
);

- RLS: Inherit from JoineryOrder (via FK)
- Indexes: (joinery_order_id, status), (plan_id, created_at desc)

### 3. Command Handler (1 day)

**File:** `Joinery.Application.Commands.GenerateAndStoreGyartasilap/`

```csharp
public record GenerateAndStoreGyartasilapCommand(
    Guid JoineryOrderId,
    Guid CuttingPlanId,
    string LabelVariant = "L1"); // default: Basic

public class GenerateAndStoreGyartasilapHandler
    : IRequestHandler<GenerateAndStoreGyartasilapCommand, GenerateAndStoreGyartasilapResponse>
{
    private readonly IRepository<JoineryOrder> _orderRepository;
    private readonly IRepository<CuttingJob> _cuttingRepository;
    private readonly GyartasilapPdfBuilder _pdfBuilder;
    private readonly IMinioClient _minioClient;
    private readonly IRepository<Gyartasilap> _gyartasilapRepository;

    public async Task<GenerateAndStoreGyartasilapResponse> Handle(...)
    {
        // 1. Fetch JoineryOrder + CuttingJobs
        var order = await _orderRepository.GetByIdAsync(request.JoineryOrderId);
        var jobs = await _cuttingRepository.GetByCuttingPlanIdAsync(request.CuttingPlanId);

        // 2. Validate: order not archived, jobs exist
        if (order.Status == OrderStatus.Archived)
            throw new InvalidOperationException("Cannot generate gyártásilap for archived order");

        // 3. Select label variant
        var variant = request.LabelVariant switch
        {
            "L1" => new LabelVariant("L1", "Basic", false, false, 10),
            "L2" => new LabelVariant("L2", "Premium", false, true, 10),
            "L3" => new LabelVariant("L3", "Barcode", true, false, 11),
            "L4" => new LabelVariant("L4", "Full", true, true, 12),
            _ => throw new ArgumentException($"Unknown variant: {request.LabelVariant}")
        };

        // 4. Generate PDF
        var pdfContent = _pdfBuilder.GeneratePdf(order, jobs.ToArray(), variant);

        // 5. Store in MinIO WORM
        var storageUrl = await _minioClient.PutObjectAsync(
            bucketName: "gyartasilap",
            objectName: $"{order.TenantId}/{request.CuttingPlanId}/gyartasilap_{variant.Code}.pdf",
            data: pdfContent,
            objectMetadata: new Dictionary<string, string> { { "version", "v1.0" } });

        // 6. Create Gyartasilap aggregate
        var gyartasilap = new Gyartasilap(
            Guid.NewGuid(),
            request.JoineryOrderId,
            request.CuttingPlanId,
            "v1.0",
            pdfContent,
            storageUrl,
            GyartasilapStatus.Draft);

        await _gyartasilapRepository.AddAsync(gyartasilap);

        return new GenerateAndStoreGyartasilapResponse(
            gyartasilap.Id,
            storageUrl,
            "Draft");
    }
}
```
