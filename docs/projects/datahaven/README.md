# Datahaven Platform

> Agent infrastruktúra és kommunikációs hub

## Vision

Központi platform az AI agent flotta koordinálásához:
- Message routing terminálok között
- Dashboard és monitoring
- Knowledge service (RAG)
- Project management

## Components

### Web Dashboard
- Real-time messaging dashboard
- Kanban board (Discovery + Delivery)
- Planning pipeline vizualizáció
- Project Hub (hierarchikus projekt kezelés)

### Telegram Bot
- Alert notifications
- Quick commands
- Status checks

### Knowledge Service
- ChromaDB vector store
- Voyage AI embeddings
- MCP integration

- **Web UI** - Dashboard, Kanban, Planning, Projects
