# SpaceOS Konsenzus Implementációs Terv

## Összefoglalás

Hibrid UI-first + backend-safety stratégia: katalógus szűrő teljes frontend implementációval (URL sync, fuzzy search, ajánlás), EHS kalkulátor gamification elemekkel, de **kritikus backend alapokkal** (audit trail). Párhuzamos fejlesztés 2 hétre, ahol Week 1 katalógus MVP + EHS backend model, Week 2 EHS UI finomítás + katalógus bővítés.

## Elfogadott prioritás sorrend

1. **Katalógus szűrő** (Week 1, napok 1-3) — frontend-only, azonnali üzleti érték, backend opcionális
2. **EHS kalkulátor backend alapok** (Week 1, napok 2-4, párhuzamos) — compliance biztonsága miatt kritikus
3. **EHS kalkulátor UI + gamification** (Week 2, napok 5-8) — user engagement növelés
4. **Katalógus ajánlás + voice search** (Week 2, napok 7-10, párhuzamos) — progressive enhancement

```javascript
catalogFilters: {
  search: '',
  category: [],
  priceRange: [0, 1000000],
  stockStatus: 'all'
},
setFilter: (key, value) => {
  set(state => {
    const newFilters = { ...state.catalogFilters, [key]: value };
    localStorage.setItem('catalog_filters', JSON.stringify(newFilters));
    // URL sync
    const params = new URLSearchParams();
    if (newFilters.search) params.set('search', newFilters.search);
    if (newFilters.category.length) params.set('cat', newFilters.category.join(','));
    window.history.pushState({}, '', `?${params.toString()}`);
    return { catalogFilters: newFilters };
  });
}
```

**Fuzzy search implementáció:**
```javascript
// useCatalogFilters.js custom hook
import Fuzzysort from 'fuzzysort';

const applyFuzzySearch = (items, searchTerm) => {
  if (!searchTerm) return items;
  const results = Fuzzysort.go(searchTerm, items, {
    keys: ['name', 'sku', 'category'],
    threshold: -10000
  });
  return results.map(r => r.obj);
};
```

**Voice search (progressive enhancement):**
```javascript
const VoiceSearchButton = () => {
  if (!('webkitSpeechRecognition' in window)) return null;
  
  const handleVoice = () => {
    const recognition = new webkitSpeechRecognition();
    recognition.lang = 'hu-HU';
    recognition.onresult = (e) => {
      setFilter('search', e.results[0][0].transcript);
    };
    recognition.start();
  };
  
  return <button onClick={handleVoice}>🎤</button>;
};
```

**Gamification rendszer (egyszerűsített):**
```javascript
const achievements = {
  safetyChampion: {
    condition: (history) => 
      history.filter(r => r.likelihoodBefore * r.severityBefore <= 6).length >= 10,
    badge: '🛡️',
    title: 'Safety Champion'
  },
  riskMitigator: {
    condition: (history) => {
      const improvements = history.filter(r => 
        r.likelihoodAfter && 
        (r.likelihoodBefore * r.severityBefore) - 
        (r.likelihoodAfter * r.severityAfter) >= 10
      );
      return improvements.length >= 3;
    },
    badge: '🎯',
    title: 'Risk Mitigator'
  }
};
```

**Risk level színlogika (bővített):**

```javascript
const getRiskLevel = (score) => {
  if (score <= 4) return { 
    color: '#10b981', // green
    label: 'Elhanyagolható',
    action: 'Nincs intézkedés'
  };
  if (score <= 9) return { 
    color: '#84cc16', // lime
    label: 'Alacsony',
    action: 'Figyelem'
  };
  if (score <= 16) return { 
    color: '#eab308', // yellow
    label: 'Közepes',
    action: 'Terv készítése'
  };
  return { 
    color: '#ef4444', // red
    label: 'Magas',
    action: 'Azonnali intézkedés!',
    alert: true
  };
};
```

## Amit Planner-A-tól veszünk át

1. **Backend-first EHS stratégia** — compliance audit trail azonnal, nem "later" fázis
2. **Közös validációs schema** — zod (frontend) + FluentValidation (backend) szinkronban
3. **Virtualizált lista** — `react-window` implementáció 5000+ katalógustétel kezelésére
4. **Konkrét kockázatelemzés** — LocalStorage limit, mobil UX, validációs drift
5. **Mermaid függőségi diagram** — vizuális development roadmap

## Amit Planner-B-től veszünk át

1. **URL state sync** — megosztható szűrő linkek (`?search=lap&cat=wood&stock=in`)
2. **Fuzzy search** — typo-toleráns keresés (`fuzzysort` lib, műhelyi használhatóság)
3. **Voice search** — progressive enhancement, kesztyűs használatra
4. **Gamification koncepció** — achievement rendszer (egyszerűsített, opt-out opcióval)
5. **Before/After slider** — UX innováció az EHS kalkulátorban
6. **Recommendation engine** — localStorage-alapú ajánlás algoritmus
7. **Export PDF funkció** — `jspdf` lib ISO 45001 audit dokumentációhoz
