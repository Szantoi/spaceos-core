## ✅ Partner KPI Widget — 2.5 nap

### Backend
**Nincs módosítás szükséges** — `sim.partners[].orders` tömb elég.

### Frontend
```
src/components/partners/
├── PartnerKpiCard.jsx         // Új komponens
│   ├── Props: partnerId, dateRange
│   ├── useEffect: calculateKPIs(orders, filters)
│   └── UI: Progress bars + trend arrows
├── KpiCalculator.js           // Pure function
│   └── onTimeRate(orders) → {percent, trend}
└── DateRangePicker.jsx        // Újrahasználható

src/pages/page-partner.jsx
└── <PartnerKpiCard partnerId={id} /> beszúrás
```

**State:** Component local state (`useState` dateRange, filtered orders).

**UX Flow:**
1. User kiválaszt partnert → KPI Card auto-load (last 30 days)
2. Date picker változik → `useEffect` újraszámol → UI frissül
3. Klikk metrikára → drill-down: filtered order table (már létező `OrdersTable` component)

**Kockázat:**
- Date picker library választás (Rolling/DayJS vs native input) → **Mitigation:** Native `<input type="date">` MVP-hez.

## ✅ QR-kód ASN Tracking — 4 nap

### Backend (MVP: Mock first)
```
Phase 1 (Mock):
src/data/mock-asn.js
└── generateASN(poId) → {asn, qrData, timestamp}

Phase 2 (Real API, +2 nap később):
POST /api/suppliers/asn/generate
├── Body: {poId, items[]}
├── Response: {asnNumber, qrCodeUrl}

POST /api/inbound/receipt/scan
├── Body: {qrData, scannedBy}
├── Response: {matched: true, poId, items}
```

**Adatmodell:**
```javascript
asn: {
  id: "ASN-20260622-001",
  poId: "PO-12345",
  supplierId: "SUP-01",
  expectedDate: "2026-06-25",
  qrHash: "sha256-abc123", // Verification
  status: "SHIPPED" | "RECEIVED"
}
```

### Frontend
```
src/components/suppliers/
├── AsnGenerator.jsx           // QR-kód modal
│   ├── qrcode.react library
│   └── Print button → PDF/Fuvarlevél
├── QrScanner.jsx              // Mobil camera input
│   ├── <input type="file" accept="image/*">
│   ├── jsQR library decode
│   └── Auto-match → ConfirmDialog
└── ReceiptConfirm.jsx         // Quantity verify

src/pages/
├── page-supplier.jsx → "Feladás" gomb
└── page-inbound.jsx → "Szkennelés" tab (új)
```

**Offline handling:**
```javascript
// LocalStorage queue
const pendingReceipts = JSON.parse(localStorage.getItem('pendingASN') || '[]');
// Background sync amikor network vissza
window.addEventListener('online', () => syncPendingReceipts());
```

**UX Flow:**
1. Supplier desktop: Feladás gomb → QR modal → Nyomtatás
2. Inbound worker mobil: Szkennel → Dekódol → "PO-12345 felismert, 20 db?" → Confirm
3. Offline: Queue → Sync later → Email notification

**Kockázatok:**
- QR library méret → **qrcode.react (7KB) választása**.
- Camera permission mobilon → **Fallback: Kézi ASN bevitel**.
- Hash collision → **UUID + timestamp alapú hash**.
