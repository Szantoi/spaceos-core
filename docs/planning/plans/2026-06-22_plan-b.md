# Planner-B — Merész Implementációs Terv

## 🎯 Stratégiai döntés: Reverse Order — KPI → QR → Autonómia

**Filozófia:** A hagyományos priorizálás adatvezéreltségre épül. De mi van, ha **először a hiányzó adatpontokat gyűjtjük be**, aztán optimalizáljuk a folyamatot?

### Fázis 1: Partner KPI Widget (1.5 nap) — Adatfeltáró munka

**Miért először?**
- **Hidden insight:** A KPI számítás során kiderül, hogy a `sim.partners[].orders` milyen minőségű. Ha sok hiányzó `delivered_date`, akkor a QR tracking **validált szükséglet**.
- **Stakeholder buy-in:** A gyártásvezető látja a piros metrikákat → azonnal támogatja a QR fejlesztést.

**Backend:**
```
GET /api/analytics/partners/:id/kpi?period=30d&status=all
Response: {
  onTimeDelivery: { value: 0.87, trend: -0.05 },
  avgLeadTime: { days: 12, trend: -2 },
  qualityRate: { value: 0.94, missing_data_count: 23 } ← KEY
}
```

**Frontend twist:**
```jsx
// widget-partner-kpi.jsx
const KPICard = ({ metric, onClick }) => (
  <div onClick={() => onClick(metric)} className="cursor-pointer hover:shadow-lg">
    {metric.missing_data_count > 0 && (
      <Badge color="orange">⚠️ {metric.missing_data_count} hiányzó adat</Badge>
    )}
  </div>
);
```

**Kockázat:** Ha a `sim` adat mock, a metrikák hamisak. **Megoldás:** Első nap valós Doorstar order history auditja.

### Fázis 2: QR ASN Tracking (3 nap) — Okos szkennelés

**Merész twist:** Ne csak ASN legyen a QR-ban, hanem **self-validating hash** (offline-first).

**Backend:**
```
POST /api/inbound/asn/generate
Request: { poId: "PO-12345", items: [...] }
Response: { 
  asn: "ASN-2026-0622-001",
  qr_payload: "ASN|PO-12345|2026-06-25|sha256_hash",
  printable_url: "/print/asn-label/..."
}

POST /api/inbound/receipt/scan
Request: { qr_payload: "ASN|...", scanned_by: "user-123" }
Response: { 
  valid: true, 
  po: {...}, 
  next_action: "QUANTITY_CONFIRM" 
}
```

**Frontend innováció:**
```jsx
// page-inbound-scan.jsx
const useCameraScan = () => {
  const videoRef = useRef();
  const [scanning, setScanning] = useState(false);
  
  useEffect(() => {
    if (!scanning) return;
    
    const scanInterval = setInterval(() => {
      const code = jsQR(getVideoFrame(videoRef.current));
      if (code?.data.startsWith("ASN|")) {
        validateOffline(code.data); // LocalStorage check FIRST
        setScanning(false);
      }
    }, 300);
    
    return () => clearInterval(scanInterval);
  }, [scanning]);
};
```

**Offline-first strategy:**
```js
// lib/offline-asn.js
const validateOffline = (payload) => {
  const [asn, po, date, hash] = payload.split('|');
  const localCache = localStorage.getItem(`asn_${asn}`);
  
  if (localCache) {
    // Instant feedback, sync later
    return JSON.parse(localCache);
  }
  
  // Compute expected hash from PO data
  const expectedHash = sha256(`${asn}${po}${date}${SECRET}`);
  if (hash === expectedHash) {
    return { valid: true, source: 'cryptographic' };
  }
  
  return { valid: false, reason: 'Invalid QR code' };
};
```

**Kockázat:** `SECRET` kezelése frontend-ben. **Megoldás:** Hash validation csak backend-en, frontend csak cache-t használ.

### Fázis 3: Autonóm Kutatás (2 nap) — Prompt Engineering Labor

**Merész twist:** Ne globális legyen, hanem **per-feature research mode**.

**Implementáció:**
```yaml
# config/research-triggers.yml
features:
  partner_kpi:
    auto_research: true
    queries:
      - "supplier scorecard best practices 2026"
      - "on-time delivery calculation methods"
    frequency: weekly
    
  asn_tracking:
    auto_research: true
    queries:
      - "QR code security manufacturing"
      - "offline-first mobile scanning UX"
    frequency: on_error  # Ha scan fail rate > 10%
```

**Backend:**
```
POST /api/research/trigger
Request: { 
  feature: "asn_tracking", 
  context: "scan_failure_rate: 0.15",
  urgency: "high"
}
Response: { 
  job_id: "res-001", 
  eta_minutes: 5,
  notification_url: "/api/research/res-001/results"
}
```

**Kockázat:** Irreleváns eredmények. **Megoldás:** Haiku confidence score + manual review flag.

## 🚀 Miért merész ez a terv?

1. **Adatminőség először** — KPI nem csak output, hanem diagnosztika.
2. **Kriptográfiai QR** — Offline biztonság hash-sel.
3. **Kontextuális kutatás** — Nem globális bot, hanem feature-assistant.

**ROI:** Ha működik, a rendszer **önjavító** lesz.
