# Chat UI — 2 bug javítása (BUG-005 Portal rész)

Az ORCH-082 (chat 422) code-fix DONE, de az AI válasz a UI-on **még nem jelenik meg** két Portal bug miatt.

## BUG-005a — Rossz BFF URL

**Fájl:** `useStreamingChat.ts` (vagy hasonló chat hook)

**Hiba:** a hook `/bff/chat` (JSON endpoint) helyett `/bff/chat/stream` (SSE endpoint)-ot kell hívnia.

```typescript
// Rossz:
const response = await fetch('/bff/chat', { ... });

// Helyes:
const response = await fetch('/bff/chat/stream', { ... });
```

## BUG-005b — SSE chunk type mismatch

**Fájl:** chat hook `applyChunk` (vagy hasonló SSE feldolgozó függvény)

**Hiba:** Az orchestrátor SSE stream-ben `{ type: 'text', text: '...' }` formátumot küld, a Portal `{ type: 'text_delta', content: '...' }` formátumot vár.

```typescript
// Jelenlegi (hibás):
if (chunk.type === 'text_delta') { append(chunk.content); }

// Helyes:
if (chunk.type === 'text') { append(chunk.text); }
```
