# BUG-014 — 404 oldal UX: "Unexpected Application Error!"

## Szimptóma

A 404 oldal generikus React error üzenetet mutat: **"Unexpected Application Error!"**

**Elvárt:** Felhasználóbarát 404 oldal (pl. "Oldal nem található" + link a főoldalra)

## Fix

Az `apps/joinerytech/src/` mappában van egy 404 / ErrorBoundary szection. Helyettesítsd:

```tsx
// Jelenleg:
<div>Unexpected Application Error!</div>

// Helyette:
<div className="flex flex-col items-center justify-center min-h-screen">
  <h1 className="text-4xl font-bold mb-4">404</h1>
  <p className="text-gray-600 mb-6">Az oldal nem található.</p>
  <a href="/" className="px-4 py-2 bg-blue-600 text-white rounded">Vissza a főoldalra</a>
</div>
```
