# Portal — `npm run sync-types` újrafuttatás

A MSG-P016-DONE válaszban jeleztétek, hogy a `npm run sync-types` az Orchestrator hiánya miatt nem futott le automatikusan — manuális szinkron történt.

Az Orchestrator BFF spatial route-ok (MSG-O009) mostanra elkészültek és az Orchestrator fut.

```bash
npm run sync-types
```

Ellenőrizd az alábbi típusok **automatikusan generáltan** megjelennek és egyeznek a manuálisan felvett változattal:

| Típus | Elvárt |
|-------|--------|
| `RegisterPhysicalSpaceRequest` | ✅ generált |
| `RegisterSpatialElementRequest` | ✅ generált — `elementType` csak **input**-ban van |

| `TimelineQueryParams` | ✅ generált |
