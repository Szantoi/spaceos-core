# FE-046 — CRM világ implementálása

## Kontextus

Új nagy sprint: a JoineryTech prototípus (https://joinerytech.hu/proto/) tartalmazza az összes tervezett világot. A Te feladatod a CRM és a Finance világ React implementálása. A prototípus a design spec — pixel-pontosan kövesd.

**Skill:** `/spaceos-terminal`  
**Sub-agent:** engedélyezett (verifier agent futtatáshoz)

## Prototípus fájlok (referencia)

```
/opt/spaceos/docs/tasks/new/joinerytech/
  page-crm.jsx        — CRM Dashboard + Pipeline + Leads + Opps + Forecast
  page-crm-2.jsx      — LeadDetail + OppDetail SlideOver + NewCrmSheet + ActivityComposer
  data-crm.js         — CRM store (mock adatok, FSM logika, engine)
  page-finance.jsx    — Finance Dashboard + Kimenő/Bejövő számlák + Kifizetések
  page-finance-2.jsx  — Számla detail SlideOver + kifizetés form
  data-finance.js     — Finance store (mock adatok, FSM)
```

## Amit implementálni kell

### CRM világ

A React app-ba (`/opt/spaceos/frontend/joinerytech-portal/src/`) add hozzá:

**1. CRM Page (`src/pages/CrmPage.tsx`)**
- Lead pipeline (kanban: Új → Kapcsolat → Minősítés → Nurturing → Konvertált)
- Opportunity lista táblázatos nézettel
- CRM Dashboard KPI-k (szabad szöveg: Pipeline érték / Win rate / Lead konverzió)
- A prototípus design language-t kövesd (stone + blue/indigo akcent)

**2. Navigáció**
- `data-worlds.js`-ben a CRM világ indigo akcent, ikon: `briefcase`
- Az App.tsx router-be bekötni (`/w/crm`)
- A HomeScreen-en megjeleníteni (WORLDS map bővítés)

**3. Mock adatok**
- A prototípus `data-crm.js` seed adatait használd referenciaként
- A React app mock/worlds struktúrájába illeszd

**4. SlideOver-ek**
- LeadDetail SlideOver (Lead adatlap, FSM gombok, aktivitásnapló)
- OppDetail SlideOver (Lehetőség adatlap, ajánlat-kapocs gomb)
- NewCrmSheet (Új lead + Új lehetőség drawer)

### Finance világ

**1. Finance Page (`src/pages/FinancePage.tsx`)**
- Kimenő számlák lista (státusz pill: draft/issued/partial/paid/void)
- Bejövő számlák lista
- Kifizetések kezelés
- Dashboard KPI: Kintlévőség / Tartozás / Lejárt

**2. Navigáció**
- `finance` világ, akcent: emerald, ikon: `briefcase`
- Router bekötés, HomeScreen kártya

**3. Mock adatok**
- A prototípus `data-finance.js` alapján

**4. SlideOver-ek**
- Számla detail (tételsorok, kifizetések, státusz FSM gomb)
- Kifizetés rögzítő form

## Tesztek

- Minden új komponenshez legalább 1 unit test
- `pnpm test` zöld kell
- CrmPage test + FinancePage test (alapvető render + navigáció)

## DONE feltételek

- Lead pipeline kanban renderel (mock adattal)
- Finance számla lista renderel (mock adattal)
- `pnpm build` hiba nélkül lefut
- `pnpm test` zöld
