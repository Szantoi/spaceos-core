## 2. Production DEPLOY

**Miután a shop világ kész és tesztek zöldek:**

```bash
cd /opt/spaceos/frontend/joinerytech-portal

# Build
pnpm build

# Deploy (a dist/ tartalmát másold a webszerverbe)
sudo cp -r dist/. /var/www/joinerytech/

# Nginx reload
sudo nginx -t && sudo systemctl reload nginx

# Smoke test
curl -s -o /dev/null -w "%{http_code}" https://joinerytech.hu/
```
