# FREETIER-FE-003 DONE — BUG-FT-003 + BUG-FT-004 fix

## BUG-FT-003: Nesting payload mismatch — FIXED

**Fájl:** `src/api/nestingApi.ts`

**Változtatások:**
1. **`{input: ...}` wrapper** hozzáadva a POST body-hoz
2. **snake_case → camelCase** konverzió: `width_mm` → `widthMm`, `height_mm` → `heightMm`, `grain_direction` → `grainDirection` (PascalCase enum: `None`/`Vertical`/`Horizontal`)
3. **`label_strategy` → `labelStrategy`**
4. **Response normalizáció:** `fromApiResponse()` function — camelCase API válasz → belső snake_case type-ok. Mindkét formátumot kezeli (camelCase és snake_case) hogy robusztus legyen.

**Megközelítés:** A belső FE type-ok/schemák maradnak snake_case-ben (formok, zod validáció, komponens props). A konverzió kizárólag az API boundary-n történik (`nestingApi.ts`). Ez minimális change, és nem töri el a meglévő 71 tesztet.

## BUG-FT-004: Magic link turnstileToken — FIXED

**Fájl:** `src/api/authApi.ts`
