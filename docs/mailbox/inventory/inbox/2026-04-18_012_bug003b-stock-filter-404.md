# BUG-003b REGRESSZIÓ — Készlet szűrő 404 (materialType param esetén)

## Szimptóma (TESTER-014, 2026-04-18 13:52)

| Kérés | Válasz |
|---|---|
| `GET /bff/inventory/stock` (param nélkül) | ✅ 200 |
| `GET /bff/inventory/stock?materialType=MDF18mm` | ❌ 404 `"Material type 'MDF18mm' not found."` |
| `GET /bff/inventory/stock?materialType=MDF16mm` | ❌ 404 |
| `GET /bff/inventory/stock?materialType=HDF3mm` | ❌ 404 |

A `0ace2f7` (space-insensitive fix) deployolva, az E2E-051 `50-inventory` 200-at adott — de az éles manuális teszten 404.

## Vizsgálandó

### 1. MaterialCatalog RLS — tenant-scoped-e?

A `MaterialCatalogs` tábla RLS policy-ja `app.current_tenant_id` alapján szűri-e a sorokat?

```sql
-- VPS-en:
SELECT * FROM spaceos_inventory.material_catalogs LIMIT 10;
-- Ha üres → nem létezik seed, vagy RLS kiszűri
SELECT relrowsecurity FROM pg_class WHERE relname = 'material_catalogs';
SELECT * FROM pg_policies WHERE tablename = 'material_catalogs';
```

### 2. Seed adat — melyik tenant_id-vel lett beillesztve?

```sql
SELECT tenant_id, material_type FROM spaceos_inventory.material_catalogs;
```

Ha `tenant_id` egy konkrét GUID → csak az a tenant látja RLS-en át.

### 3. Egyszerű reprodukálás

```bash
# test-admin tokennel közvetlenül a service-re (JWT nélkül skippel, de RLS él):
TOKEN=<test-runner-token>
curl -H "Authorization: Bearer $TOKEN" \
  "http://127.0.0.1:5004/api/inventory/stock?materialType=MDF18mm"
```

## Valószínű fix

**Ha MaterialCatalog RLS tenant-scoped, de ez reference adat:**

Az `InventoryDbContext.OnModelCreating`-ban a `MaterialCatalog` entity-re a policy-t "read-only + tenant_id IS NULL VAGY == aktuális tenant" logikával kell kiegészíteni.

VAGY: a migration seed-ben minden MaterialCatalog sort `tenant_id = NULL`-lal kell létrehozni és a RLS policy engedélyezni kell a NULL tenant_id sorokat olvasásra minden tenantnak.

**Ha az EF Core Replace() translation nem működik PostgreSQL-en:**

Cseréld le a LINQ-ban a `.Replace(" ", "") ==` összehasonlítást `EF.Functions.Like`-ra vagy explicit normalizálásra a seed adaton.
