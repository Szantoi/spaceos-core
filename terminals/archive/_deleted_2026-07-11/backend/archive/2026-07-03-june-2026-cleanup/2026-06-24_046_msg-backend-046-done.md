### API Endpoints

**PUT /api/graph/epics/:id** (`spaceos-nexus/knowledge-service/src/api/graphRoutes.ts:329-444`)

**Request body:**
```json
{
  "status": "active|done|blocked|pending",
  "depends_on": ["EPIC-ID-1", "EPIC-ID-2"],
  "parallel_with": ["EPIC-ID-3"],
  "target_date": "2026-09-30"
}
```

**Response:**
```json
{
  "success": true,
  "epic": { ... },
  "validation": {
    "valid": true,
    "cycles": []
  }
}
```

**Error responses:**
- 400 - Invalid status transition
- 400 - Cycle detected in dependencies
- 400 - Unfinished dependencies (done precondition failed)
- 404 - Epic not found
- 500 - Internal server error

### Validators

**File:** `spaceos-nexus/knowledge-service/src/graph/validators.ts` (NEW)

1. **isValidStatusTransition(currentStatus, newStatus)**
   - Validates state machine transitions
   - Rules:
     - pending → active, blocked ✅
     - active → done, blocked ✅
     - blocked → active ✅
     - done → (terminal state, no transitions) ❌

2. **validateDonePrecondition(epic, allEpics)**
   - Checks all dependencies are "done" before marking epic as "done"
   - Returns blocking dependencies if validation fails

3. **validateNoSelfReference(epicId, dependsOn)**
   - Prevents epic from depending on itself
   - Checked before cycle detection
