## Implementált funkciók

### 1. Státusz átmenet validáció (isValidStatusTransition)
**Fájl:** `spaceos-nexus/knowledge-service/src/graph/validators.ts:27-56`

State machine szabályok:
- pending → active, blocked ✅
- active → done, blocked ✅
- blocked → active (retry) ✅
- done → (terminal state, nincs továbbállás) ✅

### 2. Done előfeltétel validáció (validateDonePrecondition)
**Fájl:** `spaceos-nexus/knowledge-service/src/graph/validators.ts:58-98`

Epic csak akkor állítható "done"-ra, ha minden függősége "done":
- Ellenőrzi az összes depends_on epic státuszát
- Visszaadja a blokkoló függőségek listáját
- HTTP 400 hibával jelzi ha nem teljesül

**Teszt eredmény:**
```bash
curl -X PUT .../EPIC-GRAPH-WORKFLOW -d '{"status":"done"}'
→ {"error":"Cannot set status to done: dependencies not complete: EPIC-NEXUS-V1"}
```

### 3. Self-reference ellenőrzés (validateNoSelfReference)
**Fájl:** `spaceos-nexus/knowledge-service/src/graph/validators.ts:100-116`

Epic nem függhet önmagától:
- Ellenőrzi a depends_on listában az epic saját ID-ját
- HTTP 400 hibával jelzi ha talál

**Teszt eredmény:**
```bash
curl -X PUT .../EPIC-GRAPH-WORKFLOW -d '{"depends_on":["EPIC-NEXUS-V1","EPIC-GRAPH-WORKFLOW"]}'
→ {"error":"Epic cannot depend on itself: EPIC-GRAPH-WORKFLOW"}
```
