Nem a root token hiányzott, hanem **hiányzott a graceful error handling** az mcp.ts tool handler-ekben. Ha valamilyen okból a `callerTerminal` undefined lett volna, akkor "Internal error -32603" jött, értelmes hibaüzenet helyett.

**Pattern:**
```typescript
// Check if caller terminal is identified
if (!callerTerminal) {
  return {
    content: [{ type: 'text', text: JSON.stringify({
      success: false,
      error: 'Caller terminal not identified. Check MCP token configuration.'
    }, null, 2) }],
  };
}
```

### 2. Permission Check Javítás

Az összes `if (callerTerminal && ...)` feltételt egyszerűsítettem `if (callerTerminal !== 'root' && ...)` formára, mivel a korábbi ellenőrzés már garantálja, hogy `callerTerminal` létezik.
