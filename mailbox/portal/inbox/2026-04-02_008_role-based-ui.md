## Tárgy

Role-alapú UI + 403 console hiba megszüntetése

## Probléma

1. **Console hibák** — Designer felhasználóként navigálva az oldalak `403 Forbidden` hibákat dobnak a böngésző console-ba (pl. `/bff/api/tenants`). A TanStack Query alapból 3x újrapróbálja a hibás kérést, ami 5+ console error-t generál.

2. **Tiltott funkciók megjelennek** — Designer felhasználó látja a "Create Tenant", "Rename", "Delete" gombokat, de ezek 403-at kapnak. A felhasználó ne lásson olyan UI elemet, amit nem használhat.

## Az auth store már kész

`src/store/auth.store.ts` tartalmazza a `user.roles: string[]` tömböt, amit a JWT-ből olvas ki. Pl. Admin felhasználónál `["Admin"]`, Designernél `["Designer"]`.

## Feladatok

### 1. `useHasRole` hook — `src/hooks/useHasRole.ts`

```typescript
export function useHasRole(role: string): boolean
export function useIsAdmin(): boolean  // useHasRole('Admin') shorthand
```

### 2. TanStack Query — ne retry-oljon 4xx-re

`src/main.tsx` vagy `src/App.tsx`-ben a `QueryClient` konfiguráció:

```typescript
new QueryClient({
  defaultOptions: {
    queries: {
      retry: (failureCount, error) => {
        if (isAxiosError(error) && error.response?.status && error.response.status < 500) return false;
        return failureCount < 2;
      },
    },
  },
})
```

### 3. 403 kezelés az Axios interceptorban — `src/api/client.ts`

A 403-at ne dobja tovább unhandled errorként — vagy legyél elnéző, vagy kezeld csendesen:

```typescript
if (status === 403) {
  // Silently ignore — UI should not have shown this action
  return Promise.reject(error); // megtartjuk, de a Query retry már nem fut
}
```

### 4. Gombok elrejtése role alapján

Az alábbi helyeken csak Admin esetén jelenjenek meg a mutáló gombok:

| Oldal | Elrejtendő elem |
|---|---|
| `TenantsPage` | Create Tenant gomb, Rename + Delete akciók |
| `FacilitiesPage` | New Facility gomb, Edit + Delete akciók |
| `WorkStationsPage` | Register gomb, státusz átmenet gombok |
| `SpaceLayersPage` | New Space Layer gomb, Edit + Delete akciók |
| `FlowEpicsPage` | New Epic gomb, Delete akció |

Minta:
```tsx
const isAdmin = useIsAdmin();
// ...
{isAdmin && <button>Create Tenant</button>}
```

### 5. Read-only nézet Designernek

Ha Designer megnyit egy detail oldalt (pl. TenantDetail), jelenjen meg az adat, de ne legyenek szerkesztési gombok.
