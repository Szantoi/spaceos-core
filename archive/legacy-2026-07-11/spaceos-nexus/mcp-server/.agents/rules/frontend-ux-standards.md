# Frontend & UX Standards

## UI/UX Thinking (JTBD & Journey)
- **Jobs-to-be-Done**: Always start UI discussions by identifying the user's "Job": *"When [situation], I want [motivation], so I can [outcome]."*
- **Emotional Arc**: Consider the user's feelings (😰😕😌😊) throughout the journey map.

## Design Aesthetic (60-30-10)
- **Ratio**: 60% Primary (backgrounds), 30% Secondary (sections/cards), 10% Accent (CTAs/highlights).
- **Accessibility (WCAG AA)**:
    - Contrast: 4.5:1.
    - Touch targets: 44x44px (mobile).
    - Full keyboard navigation and screen reader support.

## Implementation
- **React**: Use Functional Components and Hooks.
- **Consistency**: Adhere to the established Design System.
- **Performance**: Optimize for speed and responsiveness.
