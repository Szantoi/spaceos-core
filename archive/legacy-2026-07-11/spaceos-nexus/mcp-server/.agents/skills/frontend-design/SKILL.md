# Frontend Design

Create production-grade interfaces that avoid "AI slop" aesthetics.

## Design Thinking
- **Purpose**: Identify the core problem.
- **Tone**: Pick a bold direction (Minimalist, Maximalist, Retro-futuristic, etc.).
- **Differentiation**: What makes it unforgettable?

## Guidelines
1. **Typography**: Characterful font choices. Pair display fonts with refined body fonts. Avoid system defaults.
2. **Color**: Cohesive, variable-based themes. Dominant colors over timid palettes.
3. **Motion**: CSS-only or Motion library for high-impact revelations.
4. **Spatial Composition**: Unexpected layouts, asymmetry, grid-breaking elements.
5. **Visual Details**: Texture, noise, gradients, and atmospheric depth.

## CRITICAL
- NEVER converge on common AI defaults (Inter/system fonts, purple gradients on white).
- Match implementation complexity to the aesthetic vision.
