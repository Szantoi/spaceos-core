# MCP Server Development Guide

## Overview
Create MCP (Model Context Protocol) servers that enable LLMs to interact with external services through well-designed tools.

## Process
1. **Research & Planning**: Understand MCP design, study specification, and framework documentation.
2. **Implementation**: Set up project structure, implement core infrastructure (auth, error handling), and develop tools with clear schemas and descriptions.
3. **Review & Test**: Verify code quality, run builds, and test with MCP Inspector.
4. **Evaluations**: Create realistic evaluation questions to test the server's effectiveness.
